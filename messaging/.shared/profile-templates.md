# Profile Templates

Templates for element profiles created by the `build-profile` command.

## Persona Profile

Location: `/messaging/personas/{slug}.md`

```markdown
# Persona: {Title}

## Overview

- **Role**: {Job title / function}
- **Type**: {Economic Buyer / Technical Decision Maker / User / Influencer}
- **Primary Pain**: {Key challenge in one sentence}

## Responsibilities

- {Core responsibility 1}
- {Core responsibility 2}
- {Core responsibility 3}

## Challenges

- {Challenge that creates urgency}
- {Challenge your solution addresses}
- {Challenge that drives buying behavior}

## Success Metrics

- {How they measure success}
- {KPIs they care about}

## Messaging Approach

- **Altitude**: {Executive / Technical Leadership / Practitioner / Developer}
- **Lead with**: {Outcome / Capability / Risk reduction}
- **Proof types**: {What evidence resonates - metrics, logos, quotes}
- **Language cues**: {Terms they use, jargon level}
```

## Segment Profile

Location: `/messaging/segments/{slug}.md`

```markdown
# Segment: {Name}

## Overview

- **Type**: {Industry / Size / Region / Maturity}
- **Defining Trait**: {What makes this segment distinct}

## Characteristics

- {Characteristic 1}
- {Characteristic 2}
- {Characteristic 3}

## Messaging Adjustments

- **Emphasis**: {What to emphasize for this segment}
- **De-emphasis**: {What matters less to this segment}
- **Proof**: {What evidence resonates}
- **Language**: {Terminology preferences}

## Relevant Personas

- {Persona 1} - {role in this segment}
- {Persona 2} - {role in this segment}
```

## Product Profile

Location: `/messaging/products/{slug}.md`

```markdown
# Product: {Name}

## Overview

- **Tagline**: {One-liner description}
- **Type**: {Core / Add-on / Module / Feature}
- **Category**: {Market category it fits}

## Job-to-be-Done

{1-2 sentences: What does this product help customers accomplish?}

## Key Capabilities

- **{Capability 1}**: {What it does and why it matters}
- **{Capability 2}**: {What it does and why it matters}
- **{Capability 3}**: {What it does and why it matters}

## Differentiation

- **Unlike**: {Alternative approach}
- **We**: {Our unique approach}

## Personas Aligned

| Persona | Why It Matters |
|---------|---------------|
| {Persona 1} | {Specific value for them} |
| {Persona 2} | {Specific value for them} |
```

## Competitor Profile

Location: `/messaging/competitors/{slug}.md`

```markdown
# Alternative: {Name}

## Overview

- **Type**: {Vendor / DIY / Status Quo / Adjacent}
- **Category**: {Their primary market category}
- **Size/Stage**: {Relevant scale info}

## Their Approach

{1-2 paragraphs: How do they position themselves? What's their strategy?}

## Our Differentiation

| Dimension | They | We |
|-----------|------|-----|
| {Dimension 1} | {Their approach} | {Our approach} |
| {Dimension 2} | {Their approach} | {Our approach} |
| {Dimension 3} | {Their approach} | {Our approach} |

## Trade-offs

- **With them**: {What customers give up or accept}
- **With us**: {What customers get instead}

## When We Win

- {Scenario where we're stronger}
- {Scenario where we're stronger}

## When They Win

- {Scenario where they're stronger - be honest}
```
